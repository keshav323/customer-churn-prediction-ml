## Project Overview

This project predicts whether a customer is likely to discontinue a company's service using Machine Learning techniques.

### Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn
- Joblib

### Machine Learning Model

- Random Forest Classifier
- Accuracy: ~77%
